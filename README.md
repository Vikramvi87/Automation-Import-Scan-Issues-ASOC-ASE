## Requirements
1 - AppScan Enterprise Server hostname
2 - AppScan Enterprise key pair
3 - Python3 for Windows (Flask and requests)
4 - Powershell 7.x

Install Python modules
```
py -m pip install Flask
py -m pip install requests
```
